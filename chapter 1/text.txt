Intro to AngularJS

Steps
- Create index.html page with HTML
- Add CDN link for angular
- Create a js folder and main.js file
- Add main.js link to index.html
- Add ng-app="app" and ng-controller="MainCtrl" to body tag
- Create divs with ng-show and ng-hide
- Add Text
- Add angular.module('app',[])
- Add .controller('MainCtrl',['$scope', function($scope){}]);
- Add $scope.user = false within MainCtrl function
- Test in browser

Goals for this post
- Able to add AngularJS to a webpage
- Create a Angular App module
- Add a Angular controller to a page
- Understand ng-show & ng-hide
